# 円盤生成アプリ（試験版）

教室の輪番管理（給食当番・係活動など）に使う「2枚重ね回転円盤」を，画面上で生成して印刷できる単一HTMLアプリです。

- 外円盤（A3） … 仕事名
- 内円盤（A4） … 児童名
- 中央を画鋲で重ねて回転させると，毎日の当番割が自動で1つズレる教室ツールです

## 特徴
- インストール不要。Edge/Chrome でURLを開くだけ
- Excel・Googleスプレッドシート・コピー&ペーストの3通りの取込みに対応
- N（分割数）は外円盤・内円盤それぞれ独立に1〜50まで
- セクションごとに塗り色・文字色・空白フラグを設定可能
- 一括設定（すべて空白／個別色を既定に戻す等）
- 「2枚を重ねてプレビュー」で実際の運用イメージを画面上で確認可
- 印刷はブラウザ標準のA3/A4印刷を自動切替
- 設定はブラウザに自動保存。JSON書出しでクラスごとに保管・翌年再利用可

## 動作環境
- Microsoft Edge / Google Chrome（PC・Chromebook）
- オフライン動作可（画像生成・印刷はローカル完結。Googleスプレッドシート取込みのみ通信が必要）
